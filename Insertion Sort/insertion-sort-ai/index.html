What is Insertion Sort?

Insertion sort builds the sorted list one item at a time, inserting each element into its proper position within the already-sorted left portion.

Algorithm intuition
  • Think of sorting playing cards in your hand: pick the next card and slot it into the correct place among the ones you already ordered.
  • Works in-place: it reorders elements inside the same array with O(1) extra space.
  • Stable: equal elements keep their original relative order.
Time Complexity
  • Worst: O(n²)
  • Average: O(n²)
  • Best (nearly sorted): O(n)
Space Complexity
  • O(1) extra space
Stability
  • Stable (keeps equal items in order)
Use Cases
  • Small arrays
  • Nearly sorted data
  • Streaming/online insertion
